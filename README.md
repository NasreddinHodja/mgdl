# mgdl

Download manga with [gallery-dl](https://github.com/mikf/gallery-dl).

## Usage

```
Usage: mgdl [OPTIONS]

Options:
  -d, --download <DOWNLOAD>
          URL of manga to download

  -a, --add <ADD>
          URL of manga to add

  -u, --update <UPDATE>
          folder name of manga to update

  -r, --reset
          Reset local DB

  -h, --help
          Print help (see a summary with '-h')

  -V, --version
          Print version
```
